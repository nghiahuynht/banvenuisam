============= cURL gửi ArgiBank=========================

API 1: Đăng nhập lấy token từ GAMAN


curl --location 'https://103.27.236.167:9005/api/User/Login' \
--header 'accept: */*' \
--header 'Content-Type: application/json-patch+json' \
--data-raw '{
  "userName": "agribank",
  "password": "Agri@667788"
}'
